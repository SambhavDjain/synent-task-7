# EduManage - Student Management System

A premium, modern web application for managing student records. Built with Vanilla Web Technologies (HTML, CSS, JavaScript), featuring a stunning glassmorphism UI.

**Live Demo**: [EduManage on Vercel](https://student-management-system-liart-chi.vercel.app)

## ✨ Features

- **Beautiful UI**: Modern glassmorphism design with a sleek dark mode palette, smooth micro-animations, and responsive layout.
- **Full CRUD Operations**: Easily Add, View, Edit, and Delete student records.
- **Persistent Storage**: Data is automatically saved to your browser's LocalStorage instantly. No databases required.
- **Import / Export**: Download your student data as a JSON file for backup, or import an existing JSON file to populate your dashboard.
- **Real-Time Search**: Filter the student list instantly by ID, Name, or Grade.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **Vanilla CSS3**: Custom properties, Flexbox/Grid layouts, glassmorphism (`backdrop-filter`), and CSS animations.
- **Vanilla JavaScript**: DOM manipulation, event handling, and LocalStorage APIs.

## 🚀 Getting Started Locally

Since the application uses standard web technologies, there's no complex setup required.

1. Clone the repository:
   ```bash
   git clone https://github.com/SambhavDjain/synent-task-4.git
   ```
2. Navigate into the folder:
   ```bash
   cd synent-task-4
   ```
3. Open `index.html` directly in any modern web browser, or serve it using a local server:
   ```bash
   # If you have Node.js installed:
   npx http-server
   
   # Or using Python:
   python -m http.server 8000
   ```

## 📂File Structure

- `index.html`: The core HTML template.
- `style.css`: All styling, theming, and layout rules.
- `app.js`: Client-side application logic and state management.
